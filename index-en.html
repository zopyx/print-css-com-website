The CSS Paged Media approach

Our PDF solutions are based on the CSS Paged Media approach. Within this approach, structured content such as XML or semi-structured HTML-based serve as the basis for the conversion. The fixing of design and layout is carried out using CSS (Cascading Stylesheets), which are also used during website programming. The CSS Paged Media standard extends CSS with print layout-specific instructions, such as the type area, the number of columns, left/right alignment, repeated text in the text in the header or footer area, etc. The advantage of our approach is that existing content can be directly reused. The print layout creation process can be mastered easily with basic knowledge in CSS.

More on the CSS Paged Media approach can be found on our portal, Print CSS Rocks.
